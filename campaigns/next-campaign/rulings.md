# 新团团桌规则 / Campaign Rulings

> **状态：已确认。** 本文件优先于新团构筑中的一般规则假设；未列出的事项仍按 D&D 2024 规则与 DM 裁定处理。

## 资源与起始构筑

| 规则 / Rule | 处理 / Application |
|---|---|
| 低价施法材料 / Low-cost spell components | 单件价值**不高于 500 gp**的法术材料，无论消耗与否，均视为免费。施法时间、法术位、法术的其他限制与固定地点限制仍然存在；价值高于 500 gp 的材料仍正常计价。 |
| 起始资金 / Starting funds | 瑟伦有 **48,000 gp** 可自由分配。现有构筑沿用原始草案的物品价格表与两件免费珍稀物品前提；若 DM 更改价格或免费物品，须重算角色卡账本。 |
| 《清晰思维之书》/ Tome of Clear Thought | 免费获得，可使用一次使智力 +2；不出售、不计入 48,000 gp，瑟伦保持人类。 |
| 神格碎片 / Divine Fragment | 每获得一枚，额外获得 1 个传奇恩惠 / Epic Boon，以及同调上限 +1。 |
| 拟像的准备法术 / Simulacrum prepared spells | 已确认：开局预制的拟像可持有与本体不同的准备法术清单，并正常施放其已准备法术。它仍不能恢复已经消耗的法术位、职业资源或其他每日资源，也不能施放拟像术。 |

## 超 20 级成长

| 规则 / Rule | 处理 / Application |
|---|---|
| 战斗胜利 / Victory | 每打赢一场战斗，角色立刻提升 1 个后 20 级成长等级。 |
| 敌人难度 / Enemy difficulty | 下一名主要敌人的 CR 比上一名主要敌人 +2；首战 CR 由 DM 设定。 |
| 准备法术 / Prepared Wizard spells | 20 级后每提升 1 个成长等级，法师可额外准备 1 个法师法术。20 级时为 25 个，因此为 `25 + 后20级成长等级`。 |
| 法术位 / Spell slots | 不再增加或解锁新的法术位；保持 20 级法师的法术位表。 |
| 兼职机会 / Multiclass opportunity | 每次后 20 级成长额外获得 1 个兼职职业等级机会，不取代法师 20 级。该兼职等级获得相应职业特性；其与“准备法术 +1”同时生效。 |

## 记录约定

- 当前角色处于 **法师 20 / 后 20 级成长 0**。首次胜利后进入成长 1，并执行角色卡中推荐的首个兼职等级。
- 熟练加值在标准规则中已于总等级 17 达到 +6；除非 DM 另行指定，后 20 级成长仍使用 +6。
- 若兼职职业自身产生准备法术，按该职业特性正常记录；“每级 +1”专门增加法师准备法术名额。
- 神格碎片的实际获得、传奇恩惠选择与新增同调物品，在获得时写入角色卡的当前状态，而非预先视作拥有。
