# Nym 推荐记录

这份文件只记录给 Nym 的玩法、升级、法术和战术建议，不作为角色事实记录。Nym 的个人状态、物品和 XP 以 [Nym 个人角色卡](../characters/nym-copperstar.md) 为准；队伍任务与剧情进展以 [KEN 小队与战役记录](../campaign/ken-party.md) 为准；已确定规则以 [团桌规则与版本裁定](../campaign/table-rulings.md) 为准。

最后更新：2026-07-10

## 当前判断

- 角色路线不需要改：继续走预言法师。
- 小队是三人：Nym 法师、蛋蛋牧师、Kael 战士。
- 小队短板：人数少、前排只有 Kael、缺少专业斥候和稳定群体处理手段。
- Nym 的定位：调查与奥术解题优先，战斗中承担控场、补伤害、保命和关键骰干预。

## 3 级升级建议

DM 已按 PHB 2024 确认，升到 3 级时法术书共新增 3 个不高于 2 环的法术：

- 普通法师升级新增 2 个法师法术。
- 预言法师额外新增 1 个预言系法师法术。

### 推荐组合

普通升级的 2 个法术：

- 蛛网术 Web：核心控场，帮助三人小队把敌人分割、定住、让 Kael 和远程攻击更容易命中。
- 粉碎音波 Shatter：补足三人队缺少群体伤害的问题，尤其适合敌人密集、需要快速清杂兵时。

额外的 1 个预言系法术首选：

- 侦测思想 Detect Thoughts：调查、审讯、社交场景价值高。

备选：定位物品 Locate Object 适合寻找黑水晶、失物或隐藏物资；看破隐形 See Invisibility 适合应对隐形敌人和异常魔法威胁。

## 3 级常备准备法术建议

3 级准备法术数会变成 6 个。护盾术 Shield 来自魔法启蒙，仍然额外可用，不占准备名额。

偏野外 / 亡灵 / 战斗：

- 法师护甲 Mage Armor
- 魔法飞弹 Magic Missile
- 蛛网术 Web
- 粉碎音波 Shatter
- 羽落术 Feather Fall
- 睡眠术 Sleep 或侦测思想 Detect Thoughts

偏城镇 / 调查 / 神殿和执法线：

- 法师护甲 Mage Armor
- 魔法飞弹 Magic Missile
- 蛛网术 Web
- 侦测思想 Detect Thoughts
- 羽落术 Feather Fall
- 睡眠术 Sleep 或粉碎音波 Shatter

备注：侦测魔法、通晓语言、鉴定术、寻获魔宠是法术书里的仪式法术时，通常不需要准备也能仪式施放；只有时间压力很大时才考虑准备。

## 魔法启蒙替换建议

当前不急着替换。

- 护盾术 Shield：保留。它是低级法师最重要的保命反应。
- 次级幻影 Minor Illusion：保留。调查、误导、潜入和创造机会都很强。
- 传讯术 Message：可留。若实际使用率很低，升级时可按本桌采用的替换规则考虑心灵裂片 Mind Sliver。

心灵裂片更偏战斗辅助，能帮后续控制法术压豁免；传讯术更偏潜入、分头行动和队伍沟通。

## 短期战术建议

- 战斗前能预判危险时，优先开法师护甲。
- Kael 顶住敌人后，Nym 不要急着追伤害，优先用蛛网术或睡眠术改变敌人行动。蛛网术锚点和物体可摧毁规则见[团桌裁定](../campaign/table-rulings.md#蛛网术-web)。
- 敌人聚集且不方便控场时，用粉碎音波。
- 对强敌或关键目标，预兆低骰优先给对方的豁免；预兆高骰优先给 Kael 的关键攻击、队友死亡豁免或关键调查检定。
- 猫头鹰魔宠主要负责侦查、Help、望风和递送触碰法术，不要让它长期停在敌人旁边。
- 三人队没有余裕硬换血，Nym 的第一原则是保持距离和维持专注。

## 4 级前后建议

4 级最稳仍然是属性提升：

- 智力 +1、体质 +1，让 INT 到 18，CON 到 16。

如果战斗中经常被打断专注，再考虑：

- 战地施法者 War Caster
- 坚韧：体质 Resilient (Con)

如果剧情调查和读心沟通越来越重要，再考虑：

- 心灵感应 Telepathic
- 敏锐头脑 Keen Mind
- 观察者 Observant
